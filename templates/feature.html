Tovice

Features

Speech or Voice Translation
In this feature the provided speech can be converted into other language.
Text Translation
The other feature is that we can also provide simple text to the application and can translate it into other language.
Text to speech
In this feature, the converted text will be converted into speech and the application will speak the converted text.
Voice Mode
In this feature, the user can use the application in voice mode totally and can translate any spoken sentence into another language.
Manual Mode
In this feature, the application gives the option to the user to operate the application in manual mode by simply typing any sentence and then choose in which language they want to translate it.
Translate from document file
If the user wants to translate the text from a particular document file the with the help of this feature he/she can easily translate it.
Save the translated text into a file
In this feature the user has given an option to save the translated text in a document file so that he/she can use it in future.
