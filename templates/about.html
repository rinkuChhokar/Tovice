Project Summary
“Tovice” is a desktop based application designed for the translation of voice from one language to another language. It is a command line based application which is executed with the help of command prompt or cmd and can be run on any other power shell or terminal in any text editor or IDE. There are mainly two modes provided by the system- Voice Mode and Manual Mode for voice translation. In the voice mode the application first ask the user to speak and automatically recognize the language and then ask in which language we want to translate it. All the process is speech recognition based here. The translated text is shown as output and we have also an option to save the translated text in a document file. The Tovice supports more than 100 different languages for translation.
